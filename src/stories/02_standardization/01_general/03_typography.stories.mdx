# Typography

## Typography component

Highpoint apps use Material-Ui's Typography component to display all text. Material-UI's default Typography style variants have been modified to match the typography styles in the Highpoint style guide.

See the [Material-Ui documentation](https://material-ui.com/api/typography/) for the full breakdown of the Typography component.
